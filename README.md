## Components:
* Arduino Nano
* DFPlayer Mini
* 3x 5V relay module (SRD-05VDC-SL-C 5V)
* 4x LED
* 4x rezistor 330ohm
* Rezistor 10kohm
* 3x diode 1N4007
* 3x tranzistor BC547
* 8x 3pin holder
* 2pin holder
